# scripts/legacy/ — historical artifacts

Scripts, logs, and reports from the v1 and v2 eras. Preserved for provenance and honest history. **Not** used in v3.0-arcadia.

- `fix_*.py`, `improve_*.py`, `retry_*.py`: ad-hoc repair scripts from v1/v2 fragility fixes.
- `vessel_orchestrator.*`: v2 phase orchestrator.
- `train_phase_*.py`: 18 v2 training phases (b, c, k, l, n, o, omega, p, q, r, s, t, u, v, w, x, y, z).
- `train_all.py`, `train_final_pipeline.py`, `train_on_real_data.py`: v1/v2 master trainers.
- `*.log`: historical training logs.
- `0.1.0`, `0.43.0`, `1.11.0`, `4.36.0`: accidental pip-list dumps from package installs.
- `training_report_v2.json`: v2 training report annotated with v3 resolutions.
